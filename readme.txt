This repository contains two basic Python programs that introduce the concepts of user input and arithmetic operations.

Contents
1. Task1.py
Description:
This script asks the user to enter two numbers. It then performs the following arithmetic operations:

Addition

Subtraction

Multiplication

Division

The results of each operation are displayed to the user.


2. task-2.py
Description:
This script asks the user to enter their first and last name. It greets the user with a personalized message using the provided names.

Run the script and follow the prompts to input your first and last name. The program will greet you using your full name.
